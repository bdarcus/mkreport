# Idea

Make it easy for university faculty to download an aggregated report of their teaching
evaluations in excel, and return nicely-formatted tables according to a template they
can add to their annual reports.

# Requirements

1. needs to run as a local storage static page, for example on github pages
2. therefore, data must be private to the local browser
3. must allow plugguble import filters (where report formats differ)
4. must have a simple template language with focus on markdown, but on the web app, the output is HTML
